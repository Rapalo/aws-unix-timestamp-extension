Features

Enable or disable extension features
Auto-detect timestamps in AWS Console
When enabled, this extension automatically detects Unix timestamps in the AWS DynamoDB Console and displays the converted date and time when you hover over them.
Show tooltips when editing items
When enabled, timestamp tooltips will be shown when hovering over timestamp fields in the item edit view.
Show time difference in tooltips
When enabled, tooltips will show how long ago or until the timestamp compared to the current time.
Show time difference in tools panel
When enabled, the tools panel will show how long ago or until the timestamp compared to the current time.
